## Agent Skills

I design and share AI Agent Skills—meta-skills to enhance and manage various non-development workflows with AI agents. Here are a few key examples:

### Git-Related Skills
- **`git-commit-flow`**: Commits AI-generated or human-edited content to git with a conventional message format (feat/fix with folder scope, prefixed with ai/human).
- **`git-diff-analyzer`**: Analyzes human edits to AI-generated content by comparing git commits, distilling patterns to iteratively improve agent skills.

### Scrum-Inspired Skills
- **`qna-file-creator`**: Implements the Flipped Interaction pattern by offloading discovery questions to a markdown file, preventing chat window token bloom during high-uncertainty phases.
- **`task-file-updater`**: Automatically maintains short session briefs to track what was done and *why* decisions were made, ensuring transparency.

For the complete collection of my workflow skills, visit the [**this repository**](https://github.com/alexeye42/workflow-skills).
